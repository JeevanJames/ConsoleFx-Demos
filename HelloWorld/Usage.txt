BACKUP /verbose /type:<backup type> /exclude=<exclude mask>,... <backup directory>

   -verbose (short: v) Provides verbose output.
   -type (short: t)    Type of backup - full or incremental. Full, if not specified.
   -exclude (short: e) One or more file masks to be excluded from the backup.
   
   backup directory    Directory to be backed up.
